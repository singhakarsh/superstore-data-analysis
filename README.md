# Superstore Sales Data Analysis

## Overview
An end-to-end data analysis project using Python and SQL 
to analyze 9994 retail sales records and extract 
business insights.

## Tools Used
- Python
- Pandas
- PostgreSQL
- SQLAlchemy
- SQL

## Key Insights
- West region has highest sales ($725K)
- South region has lowest sales ($391K)
- Canon imageCLASS Copier is #1 product ($61K)
- Technology is most profitable category ($145K)
- Texas is the state with highest losses

## Project Files
- clean_data.py - Data loading and analysis using Pandas
- load_to_db.py - Loading data into PostgreSQL
- queries.sql - SQL queries for analysis
